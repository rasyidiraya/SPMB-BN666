## TIPS PENGAMBILAN SCREENSHOT:

### 1. Persiapan Data Dummy
- Buat minimal 5 data pendaftar dengan status berbeda:
  - 1 dengan status DRAFT
  - 1 dengan status SUBMIT
  - 1 dengan status ADM_ACCEPT
  - 1 dengan status PAYMENT_PENDING
  - 1 dengan status PAYMENT_ACCEPT
- Buat 3-4 jurusan
- Buat 2-3 gelombang (1 aktif, 2 nonaktif)
- Buat user untuk setiap role

### 2. Pengaturan Browser
- Gunakan mode Incognito/Private untuk tampilan bersih
- Zoom 100%
- Sembunyikan bookmark bar
- Gunakan full screen (F11) untuk screenshot penuh

### 3. Tools Screenshot
- Windows: Snipping Tool atau Win+Shift+S
- Chrome Extension: Awesome Screenshot
- Software: Lightshot, ShareX, Greenshot

### 4. Editing (Opsional)
- Tambahkan arrow/highlight untuk tombol penting
- Blur data sensitif jika ada
- Crop bagian yang tidak perlu
- Resize jika terlalu besar (max 1920px width)

### 5. Penamaan File
- Gunakan format: `nomor_deskripsi.png`
- Contoh: `01_halaman_utama.png`
- Simpan di folder: `screenshots/`

## CARA MEMASUKKAN SCREENSHOT KE USER GUIDE:

1. Setelah semua screenshot diambil, simpan di folder `screenshots/`
2. Edit file `USER_GUIDE.md`
3. Ganti placeholder `![Screenshot: Deskripsi]` dengan:
   ```markdown
   ![Deskripsi](screenshots/folder/nama_file.png)
   ```
4. Contoh:
   ```markdown
   ![Screenshot: Halaman Utama](screenshots/pendaftar/01_halaman_utama.png)
   ```
